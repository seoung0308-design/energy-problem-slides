# 에너지 데이터로 본 현재 문제상황

워크숍용 에너지 데이터 표본을 바탕으로 만든 발표자료를 GitHub Pages에서 바로 볼 수 있도록 구성한 정적 사이트입니다.

## 구성

- `index.html`: 웹에서 바로 볼 수 있는 슬라이드 페이지
- `assets/slide-01.png` ~ `assets/slide-06.png`: 발표 슬라이드 이미지
- `assets/energy_problem_solutions_deck.pptx`: 원본 PowerPoint 파일

## GitHub Pages 설정

GitHub 저장소에 이 폴더의 파일들을 올린 뒤, 저장소 Settings → Pages에서 `Deploy from a branch`를 선택하고 `main` branch의 `/root`를 배포 대상으로 설정하면 됩니다.
